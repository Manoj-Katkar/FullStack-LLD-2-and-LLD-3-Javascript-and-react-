3. Make Important Topic Notes

Focus on the following important topics and make detailed notes:

  • Code Splitting: Dividing the codebase.
  • Dynamic Imports: Dynamically importing modules.
  • Lazy Loading: Loading components when they are needed.
  • Memoizations: Optimizing performance by caching results.

Memoization Techniques:

  • useMemo() Hook: Memoizes the result of a calculation.
  • useCallback() Hook: Memoizes callback functions to prevent unnecessary re-renders.
